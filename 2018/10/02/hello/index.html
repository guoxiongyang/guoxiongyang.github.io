如何在GitHub上搭建个人博客

一、下载Git（https://www.git-scm.com/download/win）

二、下载node.js（https://nodejs.org/en/）

在cmd中查看是否安装node成功：node -v

三、安装hexo，右击鼠标，选择Git Bash Here（hexo是基于node.js的静态博客，官网也是搭建在GitHub上）

安装hexo被墙

在 Git Bash中npm not found（重启电脑）

使用淘宝npm镜像（npm install -g cnpm –registry=https://registry.npm.taobao.org
）

使用淘宝npm安装hexo（cnpm install -g hexo-cli）

继续输入：cnpm install hexo –save

查看是否安装成功（hexo -v）

四、初始化hexo
新建一个文件夹，右击运行Git Bash，输入hexo init

初始成功后

在_config.yml中进行基础配置

查看更多主题（https://hexo.io/themes/）
使用git clone https://github.com/iissnan/hexo-theme-next themes/next来获得更多主题，主题名为next

学习一下这个操作（右击hexo文件夹）

在E:\Hexo\source_posts下新建一个 .md文件可以写文章（or 使用hexo new “hello”
）
